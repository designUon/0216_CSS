인라인 속성

block속성은 width가 부모보다 작아도 그 한 라인을 다 차지한다
inline속성은 content의 크기에 맞추기때문에 content 내용이 없으면 width, height 값이 먹히지 않는다
두개의 속성을 각각 적용할 수 있는 속성이 inline-block
그래서 block처럼 width, height 값을 가질 수 있고,
inline처럼 요소 옆에 배치 할 수 있다 => div 태그에서 많이 사용

relative 속성은 정적위치에 상대적으로 요소를 움직일 수 있다
부모의 위치를 기준으로 상대적으로 움직인다고 이야기할 수 있다

상대위치 포지션

realative를 사용할때 바로 사용을 하게 되면
원래 정적위치에서 사용한 공간만 자리를 차지하고
이동한 공간은 다른 태그가 그대로 있게 된다
*부모로 사용할 공간을 지정하고 자식태그로 relative를 사용하여 이동한다
