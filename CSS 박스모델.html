크기

크기는 font-size 속성으로 변경 할 수 있고
기본크기는 p태그의 글자크기로 동일하다.
h1태그의 글자는 이미지 200&이 된 크기로 출력이 되고있고,
동일한 font-size를 넣어주면 다른 글자들도 동일한 크기가 된다

html 태그의 전체 크기 구하기

안녕하세요

태그의 크기를 구하는 방법
contents(height or width) + padding + border + margin

box-sizing : border-box 변경

box-size를 border-box로 변경을 하고 작성하면
height / width의 값이 border + padding + content 값으로 바뀐다
크기값이 중요한 경우에는 border-box로 바꿔서 작성
크기계산이 편하기 떄문

마진값-auto

margin 속성값 auto

margin 속성값 auto를 사용해줄때는 display : block 속성이어야하고,
부모크기와 같으면 마진이 들어갈 공간이 없기 때문에
content크기가 부모보다 작아야한다
margin : auto를 사용하면 부모크기 기준의 중앙으로 정렬됨


outline

아웃라인입니다

outline은 border와 동일하게 사용할 수 있다
그려지는 위치도 동일함
단, 두께는 크기의 범위에 들어가지 않기 때문에 계산 할 필요가 없다

outline과 box-sizing:border-box로 작성했을때,
*outline을 사용하면 border값 계산X
*box-sizing을 사용하면 padding과 contents(height/width)를 따로 계산 할 필요없음
=> content 값만 계산
=> 요소의 크기 계산 : margin + content
